# AUL C-ISIT

**ケアプロフィール管理ツール — 在宅介護・障がい福祉サービス利用者のご家族向け**

> Care / Connect — It speaks for itself.

---

## これは何？

在宅介護・障がい福祉サービスを利用している方のご家族が、  
サービス情報・健康情報・緊急連絡先を**1枚のPDFにまとめて管理・共有**するためのWebアプリです。

- ログイン不要
- インストール不要
- データはあなたの端末内にのみ保存

---

## 主な機能

- 利用者情報の入力（基本情報・障がい情報・健康情報）
- 介護保険サービス・障がい福祉サービスの登録（複数）
- 通院先・担当ケアマネ・緊急連絡先の管理
- **PDFファイルとして直接ダウンロード**（担当者・病院への共有用）
- CSVでバックアップ・復元

---

## 使い方

1. [アプリを開く](https://aul-dox.github.io/AUL-C-ISIT/) ← 公開後に差し替え
2. 「プロフィールを作成する」をタップ
3. 各セクションに情報を入力（自動保存されます）
4. 「プレビューを確認する」→「💾 PDFファイルを保存」

---

## データについて

- データはブラウザのlocalStorageに保存されます（サーバーには送信されません）
- ブラウザのキャッシュを削除するとデータが消えます
- 定期的に「保存（CSV）」でバックアップしてください
